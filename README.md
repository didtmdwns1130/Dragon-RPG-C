# Dragon RPG

C 언어로 구현한 콘솔 기반 성장형 RPG 미니게임입니다.  
플레이어가 드래곤을 조작하여 몬스터를 처치하고 레벨업하는 구조를 가지고 있습니다.

## 주요 기능
- 플레이어 이동
- 몬스터 등장 및 전투
- 체력 시스템
- 경험치 및 레벨업
- 간단한 콘솔 UI 구성

## 개발 목적
- C 언어 문법 및 제어 구조 학습
- 콘솔 기반 게임 로직 설계 연습
- 반복문, 조건문, 배열 등 기초 로직 강화

## 파일 설명
- **Dragon-RPG.c** : 메인 게임 로직이 구현된 코드 파일

## 실행 환경(Environment)

이 프로젝트는 아래 환경에서 개발 및 실행되었습니다.

- **운영체제:** Ubuntu 24 LTS
- **컴파일러:** GCC (GNU Compiler Collection)
- **터미널:** 기본 Bash 터미널

**주의:**  
Windows CMD / PowerShell 환경에서는  
문자 인코딩, 콘솔 출력 방식 등의 차이로 인해  
정상적으로 동작하지 않을 수 있습니다.  

Linux(특히 Ubuntu) 환경에서 실행하는 것을 권장합니다.

## 실행 방법
1. GCC 컴파일러가 설치되어 있는지 확인합니다.

2. 아래 명령어로 코드를 컴파일합니다:
   gcc Dragon-RPG.c -o dragon

3. 실행:
   ./dragon

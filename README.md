# Dragon RPG (C)

C 언어로 구현한 콘솔 기반 성장형 RPG 미니게임입니다.  
플레이어가 드래곤을 조작하여 몬스터를 처치하고 레벨업하는 구조를 가지고 있습니다.

## 주요 기능
- 플레이어 이동
- 몬스터 등장 및 전투
- 체력 시스템
- 경험치 및 레벨업
- 간단한 콘솔 UI 구성

## 개발 목적
- C 언어 문법 및 제어 구조 학습
- 콘솔 기반 게임 로직 설계 연습
- 반복문, 조건문, 배열 등 기초 로직 강화

## 파일 설명
- **Dragon-RPG.c** : 메인 게임 로직이 구현된 코드 파일

## 실행 방법
1. C 컴파일러 설치  
2. 코드 컴파일
